Google Analyticsで効果測定しよう

LPを公開したら、次は「どれくらい見られているか」を知ろう

効果測定とは

LPを公開しただけでは、誰が見てくれたのか、どれくらいアクセスがあったのかはわかりません。効果測定とは、公開したページの成果を数字で把握することです。

たとえば、こんなことがわかります：

  • 何人がページを見たか — アクセス数（ページビュー）
  • どこからたどり着いたか — SNS経由？検索経由？URLを直接入力？
  • どれくらいの時間見ていたか — すぐ離脱されたか、じっくり読まれたか
  • スマホとPC、どちらが多いか — デバイスの比率

なぜ効果測定が大事？

数字を見ることで「SNSでシェアしたら一気にアクセスが増えた」「スマホからの閲覧が8割だからスマホ表示を優先しよう」といった判断ができます。感覚ではなくデータで次の改善を決められるのが効果測定のメリットです。

効果測定にはさまざまなツールがありますが、最も広く使われているのがGoogle Analytics（グーグル アナリティクス）です。Googleアカウントがあれば無料で使えます。

Google Analyticsのタグを埋め込もう

Google AnalyticsでLPのアクセスを計測するには、HTMLに計測用のタグ（短いコード）を埋め込む必要があります。タグを埋め込む作業はClaudeに任せられます。

ステップ1：Google Analyticsのアカウントを作成する

  1. ブラウザで analytics.google.com にアクセス
  2. Googleアカウントでログインする
  3. 「測定を開始」をクリック
  4. アカウント名を入力（自由な名前でOK。例：「LP効果測定」）
  5. プロパティ名を入力（サイトの名前。例：「カフェLP」）
  6. タイムゾーンと通貨を「日本」「日本円」に設定
  7. ビジネスの情報を入力して進む

ステップ2：データストリームを設定する

  1. プラットフォームで「ウェブ」を選択
  2. ウェブサイトのURLに https://corklab.github.io/tutorial を入力
  3. ストリーム名を入力（例：「LP」）
  4. 「ストリームを作成」をクリック

ステップ3：測定IDを確認する

ストリームが作成されると、測定ID（Measurement ID）が表示されます。

G-XXXXXXXXXX

「G-」から始まるこのIDをコピーしてください。

ステップ4：ClaudeにタグをHTMLに埋め込んでもらう

コピーした測定IDをClaudeに渡して、タグの埋め込みを依頼します：

あなた → Claude

「LPのHTMLにGoogle Analyticsのタグを埋め込んで。測定IDは G-XXXXXXXXXX です。コミットとプッシュまでやっておいて」

Claudeは以下のようなコードをHTMLの <head> 内に自動で追加してくれます：

<!-- Google tag (gtag.js) --> <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script> <script> window.dataLayer = window.dataLayer || []; function gtag(){dataLayer.push(arguments);} gtag('js', new Date()); gtag('config', 'G-XXXXXXXXXX'); </script>

あとはいつもどおり、ブラウザでPRを作成してマージすれば完了です。

ポイント

タグを埋め込んだ直後からデータの収集が始まりますが、Google Analyticsの画面に反映されるまで最大48時間ほどかかる場合があります。すぐにデータが見えなくても心配ありません。

Google Analyticsの画面の見方

タグの埋め込みとデプロイが完了したら、analytics.google.com にアクセスして計測データを確認できます。

リアルタイムレポート

左メニューの「レポート」→「リアルタイム」を開くと、今この瞬間にサイトを見ている人数が表示されます。タグを埋め込んだ直後の動作確認にも使えます。自分でLPにアクセスして、リアルタイムに「1」と表示されれば、タグが正しく動いている証拠です。

ユーザー属性

「レポート」→「ユーザー属性」→「概要」を開くと、訪問者の情報がわかります：

項目わかること
国どの国からアクセスがあるか
市区町村おおまかな地域
言語ブラウザの表示言語

テクノロジー

「レポート」→「テクノロジー」→「概要」を開くと、訪問者の環境がわかります：

項目わかること
プラットフォームPC / スマホ / タブレットの比率
ブラウザChrome / Safari / Edge など
OSWindows / Mac / iOS / Android など

スマホ比率が高かったら？

Claudeに「スマホでもっと見やすくして」と頼みましょう。データに基づいた改善ができるのが効果測定の強みです。

集客（トラフィック獲得）

「レポート」→「集客」→「トラフィック獲得」を開くと、訪問者がどこから来たかがわかります：

チャネル意味
DirectURLを直接入力、またはブックマークからアクセス
Organic SocialX（Twitter）、Instagram、FacebookなどSNSからのアクセス
Organic SearchGoogleなどの検索結果からのアクセス
Referral他のサイトに貼られたリンクからのアクセス

エンゲージメント

「レポート」→「エンゲージメント」→「概要」を開くと、訪問者がどれくらいページに関わったかがわかります：

指標意味
平均エンゲージメント時間ページを実際に見ていた平均時間
エンゲージメント率10秒以上滞在 or 2ページ以上閲覧した人の割合
表示回数ページが表示された合計回数

まずはこれだけ見ればOK

Google Analyticsには膨大な機能がありますが、LP運用で最初に見るべきは「リアルタイム」（タグの動作確認）と「集客」（どこからアクセスが来ているか）の2つです。慣れてきたらユーザー属性やエンゲージメントも活用していきましょう。
