# MacのClaudeにお任せ・ブラウザ操作だけで完結するLP公開チュートリアル

プログラミング経験ゼロでも大丈夫。MacのClaudeアプリに技術的な作業をすべて任せて、あなたはブラウザを操作するだけでランディングページ（LP）を世界に公開できます。

## このチュートリアルで学べること

- GitHubでリポジトリを作る方法
- MacのClaudeにコードを書いてもらう方法
- GitHub Pagesで無料でWebページを公開する方法
- LPを更新するサイクルの回し方

## 前提条件

- Mac を持っていること
- [Claude](https://claude.ai/download) のデスクトップアプリがインストールされていること
- [GitHub](https://github.com) のアカウントを持っていること（無料でOK）
