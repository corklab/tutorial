MacのClaudeにお任せ
ブラウザ操作だけで完結する
LP公開チュートリアル

プログラミング経験ゼロでも大丈夫。管理者から招待されたリポジトリに、MacのClaudeアプリを使ってLPを作成・公開できます。あなたはブラウザとClaudeに話しかけるだけ。

このチュートリアルで学べること

  • 管理者からの招待を受けてリポジトリに参加する方法
  • MacのClaudeにコードを書いてもらう方法
  • プルリクエストを作ってマージする方法
  • LPを更新するサイクルの回し方

前提条件

💻 Mac を持っていること
🤖 Claude のデスクトップアプリがインストールされていること
🐙 GitHub のアカウントを持っていること（無料でOK）
✉️ 管理者からリポジトリへの招待メールが届いていること

全体の流れ

あなた（コラボレーター）と管理者の役割分担

管理者：リポジトリ作成 → 招待 → Pages公開設定（最初のセットアップだけ）
あなた：招待を受ける → Claudeと一緒にLPを作る → PRを出してマージ → 公開！

目次

  • 第1話 招待を受けてリポジトリに参加する
    コラボレーター招待を承認し、PRが出せる状態を確認する
  • 第2話 Claudeがコードを書く・安全な作業部屋でのLP作成
    ブランチを作り、HTMLを書いてもらう
  • 第3話 コミットも自動・ClaudeがMacでセーブポイントを記録
    ファイルの追加とコミットを任せる
  • 第4話 Pushも丸投げ・GitHubへデータを打ち上げる
    プッシュしてプルリクエストを作る
  • 第5話 世界へ公開・マージしてLPを本番反映
    PRをマージしてGitHub PagesでLPが公開されるのを確認する
  • 第6話 修正も一瞬・Claudeと回す更新サイクル
    修正→PR→マージ→公開の流れをマスターする

コラム

  • コラム GitHub Pagesでつくる無料ホスティングのLP公開
    GitHub Pagesの仕組み・できること・留意事項・独自ドメイン設定
  • コラム Google Analyticsで効果測定しよう
    効果測定とは・タグの埋め込み方・画面の見方

所要時間

全6話で約30〜45分です。1話ずつ区切って進められます。
