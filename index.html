MacのClaudeにお任せ
ブラウザ操作だけで完結する
LP公開チュートリアル

プログラミング経験ゼロでも大丈夫。MacのClaudeアプリに技術的な作業をすべて任せて、あなたはブラウザを操作するだけでランディングページ（LP）を世界に公開できます。

このチュートリアルで学べること

  • GitHubでリポジトリを作る方法
  • MacのClaudeにコードを書いてもらう方法
  • GitHub Pagesで無料でWebページを公開する方法
  • LPを更新するサイクルの回し方

前提条件

💻 Mac を持っていること
🤖 Claude のデスクトップアプリがインストールされていること
🐙 GitHub のアカウントを持っていること（無料でOK）

目次

  • 第1話 GitHubでリポジトリ作成とClaudeへのクローン指示
    GitHubで箱を作り、Claudeに引き継ぐ
  • 第2話 Claudeがコードを書く・安全な作業部屋でのLP作成
    ブランチを作り、HTMLを書いてもらう
  • 第3話 コミットも自動・ClaudeがMacでセーブポイントを記録
    ファイルの追加とコミットを任せる
  • 第4話 Pushも丸投げ・GitHubへデータを打ち上げる
    プッシュしてプルリクエストを作る
  • 第5話 世界へ公開・GitHub Pagesのスイッチオン
    マージしてPagesを有効化する
  • 第6話 修正も一瞬・Claudeと回す更新サイクル
    修正→公開の流れをマスターする

所要時間

全6話で約30〜45分です。1話ずつ区切って進められます。
