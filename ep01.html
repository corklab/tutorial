第1話

招待を受けてリポジトリに参加する

目標：コラボレーター招待を承認し、PRが出せる状態を確認する

この話の流れ

管理者 あなた（ブラウザ） Claude（Mac） | | | |-- 招待メール送信 --->| | | |-- 招待を承認 | | |-- リポジトリURL確認 | | |-- 「クローンして」--->| | | |-- クローン実行 | | |-- 準備完了

実践ステップ

ステップ1：招待メールを確認する

管理者がリポジトリにあなたを招待すると、GitHubに登録しているメールアドレスに招待メールが届きます。

  1. メールの件名は「[corklab/tutorial] You've been invited...」のような形式です
  2. メール内の「View invitation」ボタンをクリック

メールが見つからない場合

ブラウザで直接 https://github.com/corklab/tutorial/invitations にアクセスしても招待を確認できます。また、GitHubの通知（ベルアイコン）にも表示されます。

ステップ2：招待を承認する

  1. GitHubの招待ページが開く
  2. 「Accept invitation」ボタンをクリック
  3. リポジトリのページに移動すれば成功

ステップ3：リポジトリにアクセスできることを確認する

承認後、リポジトリのページが表示されます。URLはこの形式です：

https://github.com/corklab/tutorial

このページが見えていれば、あなたはこのリポジトリにプルリクエストを出せる状態です。

ステップ4：Claudeにクローンを指示する

MacのClaudeアプリを開いて、リポジトリのURLを渡します：

あなた → Claude

「このGitHubリポジトリをMacにクローンして、LP作成の準備をして」

https://github.com/corklab/tutorial

ステップ5：Claudeの動きを見守る

Claudeが以下を自動で実行します：

  1. Mac上の適切な場所にフォルダを作成
  2. git clone コマンドでリポジトリをダウンロード
  3. 作業フォルダの準備完了を報告

ポイント

Claudeがターミナルコマンドを実行するとき、許可を求めてくることがあります。内容を確認して「Allow」を押してください。

この話のまとめ

やったこと誰が
リポジトリ作成・招待送信管理者
招待を承認あなた（ブラウザ）
アクセス確認あなた（ブラウザ）
クローン実行Claude（Mac）

これで準備完了。次の話から、実際にLPを作っていきます。
